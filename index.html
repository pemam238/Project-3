Interactive Data Visualization Project

Exploring patterns through D3 interaction, filtering, and tooltips

Visualization

This interactive graphic allows users to explore relationships in the dataset using visual encodings, filters, and details-on-demand.

Design Rationale

I chose a scatterplot because it is effective for comparing two quantitative variables. Position on the x- and y-axes makes it easy to identify trends, clusters, and outliers. Color is used to separate categories when available, while tooltips provide extra detail without overcrowding the chart.

The interaction techniques include dropdown menus for changing the displayed variables and tooltips for details-on-demand. These interactions help users explore multiple relationships in the dataset instead of only seeing one static view.

Data Transformations

The data is loaded from a static CSV file. Numeric columns are converted from strings into numbers so they can be accurately plotted. Missing or invalid values are filtered out before drawing the chart.

Development Process

The project was developed by first exploring the dataset, then choosing the most meaningful variables to compare. Most of the development time was spent cleaning the data, setting up scales and axes, and making the interactions work smoothly.
